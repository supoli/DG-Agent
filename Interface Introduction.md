# 界面说明
>此处为web界面的说明，Android还在测试阶段。

## 界面从上到下依次为：状态栏、对话区、输入区。


**状态栏：**
-

*左部分*
- 历史对话记录。
- 指示灯：当主机与web连接成功时，绿灯亮起，否则圆圈为灰色。
  
*中部分*
- 模型风格（场景模式）：初始默认为”温柔调情“，点击模型风格按键进入，弹出“场景模式”选择区域，可见web自带六个模型风格，分别为：温柔调情、主导调教、欲擒故纵、奖惩游戏、边缘控制、温情陪伴。各初始模型风格描述见web。
- 自定义模型人设：在“场景模式”选择区域下方，是自定义模型人设的选项，点击该选项，在弹出的输入框中输入想要的ai人设和互动风格，输入完成后，点击右下方的“保存”。

*右部分*
- 紧急停止：点击红色的停止按键可以停止输出。
- 蓝牙配对：点击蓝牙图标开始扫描已开启蓝牙的设备，要求浏览器拥有蓝牙功能。扫描到郊狼设备后，点击配对，查看郊狼主机指示灯是否由黄变白，变白则配对成功。
- 设置：点击齿轮图标进入设置，进入设置后可以看到上方三个选项：基础设置、安全，和波形。
1.基础设置
（1）主题模式：选择深色或是浅色，或是跟随系统。
（2）模型选择：
除了开发者提供的免费模型外，此处可以通过对支持的四种模型的api与模型版本配置来使用这些模型，或在此基础上增加url以自定义一个自己使用的模型。
2.安全
在安全界面，设置A、B两个通道各自的强度上限；并确认工具调用模式；另外，设置后台行为，关闭这个默认打开的选项，可以使该浏览器界面·处于后台时仍然连接郊狼主机并输出，否则，在该浏览器界面切至后台时将切断输出和与主机的连接，需要重新回到界面进行蓝牙连接和与模型对话来启动输出。
3.波形
这个界面支持对自带的六种波形进行自然语言描述式的调整，另外，可以通过点击“导入”按键，选中.pulse文件导入波形，为ai性能考虑，不建议总波形数量大于10个。

**对话区：**
-
此处显示了您和ai的对话。

**输入区：**
-
通过在此区域键入消息，并通过点按界面左下角的“飞机”键发送消息实现和ai的对话，并通过对话实现开始郊狼输出和输出的调整。
